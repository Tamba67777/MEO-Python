# Expense Tracker - Трекер расходов

## Автор
**Имя и Фамилия:** Астафьев Роман

## Описание программы
Expense Tracker - это простое графическое приложение для отслеживания личных расходов. Программа позволяет:
- Добавлять расходы с указанием суммы, категории и даты
- Просматривать список всех расходов в удобной таблице
- Фильтровать расходы по категории и временному периоду
- Автоматически подсчитывать сумму расходов за выбранный период
- Сохранять и загружать данные в формате JSON
- Удалять ненужные записи

## Требования
- Python 3.7 или выше
- Установленные библиотеки:
  - `tkinter` (входит в стандартную поставку Python)
  - `tkcalendar` (требуется установить)
